URL Shortener Microservice

User Story:

  1. You can POST a URL to [project_url]/api/shorturl/new and You will receive a shortened URL in the JSON response.
  2. When You visit the shortened URL, it will redirect You to Your original link.

Short URL Creation

example: POST [project_url]/api/shorturl/new - https://www.google.com

Example Usage:

[this_project_url]/api/shorturl/3

Will Redirect to:

https://forum.freecodecamp.com
